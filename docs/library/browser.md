### R.isChorme

> 是否为Chorme浏览器

```javascript
R.isChorme()
```

**参数：**

​	无

**返回值：**

​	`boolean`

​	true: 是 | false: 否

 **例子：**

```javascript
isChorme()
// => true or false
```
