# 字符相关

# API

名称 描述
camelCase 字符串转驼峰
guid 随机UUID
randomHexColorCode 随机十六进制颜色

# R.camelCase

字符转为大驼峰，

R.camelCase(value)

参数：

​ 1. value {string} 需要转换的字符

返回值：

​ string

​ 返回大驼峰字符串

例子：

camelCase("hello_word")
// => "HelloWord"

camelCase("welcome_to_chengdu")
// => "WelcomeToChengdu"

# R.guid

随机生成36位字符ID, 类似UUID

R.guid()

参数：

​ 无

返回值：

​ string

​ 36位随机字母、数字、下划线组成的id，如： db9b82c8-784a-6d3d-b4f1-2e312c79071a

例子：

guid()
// => "db9b82c8-784a-6d3d-b4f1-2e312c79071a"

# R.randomHexColorCode

随机十六进制颜色

R.randomHexColorCode()

参数：

​ 无

返回值：

​ string

​ 随机十六进制颜色，如：#0099ff

例子：

randomHexColorCode()
// => "#0099ff"
