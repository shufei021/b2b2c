# R.add

两数或多数相加的和

R.add(value1, value2, value3)

参数：

​ 1.value1 {number | array} 加数，如果需要多个可传入 array 进行相加；

​ 2.value2 {number} 被加数， 如果第一个参数为 array，那么第二个参数则为保留小数位数，第三个参数可不传；

​ 3.value3 {number} 结果保留位数；

返回值：

​ number

​ 计算后的和

例子：

// 两数相加
add(1.325, 1.235, 2)
// => 2.56

// 多数相加，第一个参数为数字list，第二个则为保留位数
add([1, 2, 3, 4, 5, 6, 7.124], 2)
// => 28.12

# R.isNum

是否有效数字：排除 NaN 、 Infinity，数字字符串或数字，都为有效数字

R.isNum(value)

参数：

​ 1.value {number} 检测的值

返回值：

​ boolean

​ true: 是 | false：否

例子：

let arr = [{}, [], undefined, null, 123, '123', Symbol('111'), function () {}, new Date(), false, /\d+/gi, NaN,Infinity]
arr.forEach(item => {
  console.log(isNum(item))
  // true: (123, '123')
  // false: (other)
})

# R.isInteger

是否为整数, 必须是数字类型

R.isInteger(value)

参数：

​ 1.value1 {number} 检测的值

返回值：

​ boolean

​ true: 是 | false：否

例子：

isInteger(3)
// => true

isInteger(3.11)
// => false

isInteger('3')
// => false

isInteger(NaN)
// => false
